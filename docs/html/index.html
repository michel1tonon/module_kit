module_kit Documentation

Practical documentation for modular composition in Flutter.

This library does not do anything by itself. You still need to wire dependency injection and routes in the app shell, otherwise it will not work.

This lib does not do magic, but remember: every magic has a price. :)

Philosophy

  • Modular composition with small and predictable contracts.
  • Supports micro frontends and multi-package setups, without forcing architecture.

Design Decisions

  • Freedom over rigid conventions.
  • FeatureModule contributes routes and injectors.
  • ModuleComposer aggregates enabled modules.
  • ModuleComposerBuilder is optional.
  • CrossDependency encourages self-contained modules.

  • Team expects a fully opinionated framework with everything prewired.
